Website Monitoring System - n8n Workflow
**Description**
Automated website monitoring workflow built with n8n that checks website status every 5 minutes and sends email alerts when the website goes down.
**Features**
Monitors website every 1 minutes
Instant email alert when website is down
Automatic status check (UP/DOWN)
Built for Tanzeela Fresh Market website

**Tools Used**
- **n8n** - Workflow automation
- **GitHub Pages** - Website hosting
- **Gmail** - Email alerts

## Workflow Steps
1. Schedule Trigger (every 5 minutes)
2. HTTP Request (check website status)
3. IF condition (status 200 = UP, else DOWN)
4. Email alert via Gmail
